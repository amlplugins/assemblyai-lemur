# @amlplugins/assemblyai-lemur

Native `assemblyai` SDK re-exported under the `@amlplugins` namespace.

> AssemblyAI LeMUR — audio-grounded LLM for Q&A, summary, action items, and custom prompts over transcripts.

## Install

```bash
npm install @amlplugins/assemblyai-lemur
```

## Use

```ts
import { /* named exports from assemblyai */ } from "@amlplugins/assemblyai-lemur";
// or
import sdk from "@amlplugins/assemblyai-lemur";
```

Built and published by GitHub Actions on every push to `main`.

Part of the amlplugins family — see https://github.com/amlplugins.
